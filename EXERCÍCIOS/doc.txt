Listas mistas e de definição

listas:

LISTA NÃO ORDENADA.

ul = unordered list.
Ex:
<html>
<ul>
    <li>café</li>
    <li>Almoço</li>
    <li>Jnata</li>
</ul>

<web>
. café
. Almoço
. Janta
Obs: o ponto antes do ítem é o tipo padrão da "ul", também podemos definir outros dois, o Circle(um círculo branco) e Square(um quadrado preto) utilizando código abaixo:
<html>
<ul type="clircle"></ul>
<ul type="square"></ul>



LISTA ORDENADA

ol = ordered list.( acrescenta numeração, letras ou algarismo romano para enumerar os itens.)
Ex:
<html>
<ol>
    <li>Cliper</li>
    <li>visual Basic</li>
    <li>Fortran</li>
    <li>Delphi</li>
</ol>
<web>
1. Cliper
2. visual Basic
3. Fortran
4. Delphi
Obs: As listas ordenadas atribuem por padrão, numeração aos ítens, mas também podemos definir os tipos: a-A(alfabético), i-I(Romano) utilizando o seguinte código:
<html>
<ol type="a">
    <li>Cliper</li>
    <li>visual Basic</li>
    <li>Fortran</li>
    <li>Delphi</li>
</ol>

<web>
a. Cliper
b. visual Basic
c. Fortran
d. Delphi

Além do "type", podemos atribuir o "start" que faz a lista começar a partir de um número definido.
Ex:
<html>
<ol>
    <li>Antigas</li>
    <ol type="a">
        <li>Cliper</li>
        <li>visual Basic</li>
        <li>Fortran</li>
        <li>Delphi</li>
    </ol>
    <li>Novas</li>
    <ol type="a" start="5"
        <li>PHP</li>
        <li>Python</li>
        <li>JavaScript</li>
        <li>Ktlin</li>
</ol>

<web>
1. Antigas
    a. Cliper
    b. visual Basic
    c. Fortran
    d. Delphi
2. Novas
    e. PHP
    f. Python
    g. JavaScript
    h. Ktlin
Os ítens da "ol novas" começam a partir da letra "E" que é a quinta letra do alfabeto. O start é EXCLUSIVAMENTE numérico ainda que o "type" seja alfabético ou romano.

EXE-011
IMAGENS DINÂMICAS


Com html utilizaremos 3 fotografias de tres tamanhos diferentes. Cada foto será mostrada dependendo do tamanho do dispositivo. para isso utilizaremos a tag <picture>
Ex:
<picture>
    <source media="(max-width: 750px)" srcset="imagens/foto-p.png" type="image/png">
    <source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png">
    <img src="./imagens/foto-g.png" alt="imagem flexível">
</picture>

Passo a passo: 
Usando a tag <img> e informando o endereço, vamos incluir primeiramente a foto maior que será exibida na tela maior. Depois vamos incluir acima da primeira, a próxima menor foto utilizando a tag <source: media: type> e atribuindo "(max-width: 777px)", o endereço da imagem e o "type"







REPRODUZINDO AUDIO.


Onde conseguir as músicas
youtube-painel de controle-youtube studio-biblioteca de audio- atriubição não necessária- usar filtros- baixar.
Os principais arquivos aceitos pelos navegaroes são:
1°- mp3 
2°- wav (Evitar o formato devido ao tamanho)
3°- ogg

Para inserir arquivo de audio, utilizaremos a tag "<audio>" 
Ex-01:

<audio src="endereço do arquivo" controls(para aparecer os comandos de play,stop,volume...) ></audio>

Ex-02:

<audio preload="metadata" autoplay controls loop>
    
    <source src="./midia/White River - Aakash Gandhi.mp3" type="audio/mpeg" >
    
    <source src="./midia/White River - Aakash Gandhi.ogg" type="audio/ogg">
    
    <source src="./midia/White River - Aakash Gandhi.wav" type="audio/wav">
    
    <p>Infelismente seu navegador não consegue reproduzir àudio. <a href="./midia/White River - Aakash Gandhi.mp3">Clique aqui para baixar o arquivo</a></p>
</audio>

Desa forma o código vai tentar abrir cada formato até encontrar um que funcione e caso nenhum funcione, abrirar o link para download do arquivo.
Obs:
    1- O atributo "preload" possue 3 valores: auto, metadata e none
    Auto = ele só vai considerar que o site carregou depois de carregar todo o audio

    metadata = Vai carregar apenas os metadados. nome do arquivo, formatos...

    none = não carrega nada. Só começa quando o usuário apertar o play

    2- O atributo autoplay não funciona pelo google chrome

    3- O atributo "loop" faz o audio recomeçar quando chega ao final.

https://www.youtube.com/watch?v=fzyab4P2pn8&list=PLHz_AreHm4dkZ9-atkcmcBaMZdmLHft8n&index=39

ao inves de inserir o CSS em cada linha, nós iremos criar um <style> dentro do <head> e dentro dele iremos criar chamar os seletores.
Ex:
<head>
    <style>
        body {
            background-color: aqua;
            font-family: Arial, Helvetica, sans-serif;
        }
        h1{
            color: blue;
            background: rgba(1, 1, 255, 0.418);
        }
        h2{
            color: darkred;
        }
        p{
            text-align: justify;
        }
    </style>
    
</head>
<body>
    <h1>Capítulo 1</h1>
    <h2>Cpítulo 1.1</h2>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Eaque, maxime inventore! Voluptate, totam! Dolorum ut iusto et, ipsa provident distinctio minima eaque nihil doloremque commodi, quaerat tempora facere nam modi.</p>

    <h2>Capítulo 1.2</h2>
    <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Accusantium enim alias saepe, odio quod dicta. Odit veniam modi cumque, officiis sint aliquid, expedita nihil delectus fugit non recusandae corrupti minima.</p>
    
    <h1>Capítulo 2</h1>
    <h2>Capítulo 2.1</h2>
    <p>Lorem ipsum dolor sit, amet consectetur adipisicing elit. Neque ab in molestiae ut recusandae minima ipsum sequi officiis error obcaecati totam inventore, corrupti aspernatur pariatur natus porro explicabo repellendus animi.</p>
</body>


Obs:
1°- Vantagem:
 A alteração se aplica a todas as tags de uma única vez.

2°- Desvantagem:
 O estilo fica muito grande, atrapalhando chegar no html.

3°- Desvantagem:
 Se for preciso utilizar em outras páginas, será preciso copiar e colar o estilo em cada uma.

https://www.youtube.com/watch?v=-i1JVMspDJQ&list=PLHz_AreHm4dkZ9-atkcmcBaMZdmLHft8n&index=40

O mais indicado método para estilo é criando um documento CSS atravé da tag <link> no <head>.

Ex:
<html>
<link rel="stylesheet" href="style(ou qualquer outro título que voce escolheu para sua pasta CSS).css">
</html>
depois cria-se uma arquivo CSS onde iremos inserir todos os estilos que serão soliciados para o "html" atravé do "<link>"

Ex:
<CSS>

@charset "UTF8";
body {
    background-color: lightgoldenrodyellow;
    font-family: Arial, Helvetica, sans-serif;
}
h1{
    color:  maroon;
    background: darkgoldenrod
}
h2{
    color: darkred;
}
p{
    text-align: justify;
}

Vantagem:
todos os estilos são aplicados aos elementos em todas as paginas e ainda é possível utilizar CSS interno e CSS inline  para páginas e elementos específicos.
Ex:
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Estilos Externos</title>
    <link rel="shortcut icon" href="favicon.icon" type="image/x-icon">
    <link rel="stylesheet" href="style.css">
    <style>
        h1{
            text-decoration: underline;
        }
    </style>
    <!--Estilo aplicado apenas na página em questão-->
</head>
<body>
    <h1>Capítulo 1</h1>
    <h2>Cpítulo 1.1</h2>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Eaque, maxime inventore! Voluptate, totam! Dolorum ut iusto et, ipsa provident distinctio minima eaque nihil doloremque commodi, quaerat tempora facere nam modi.</p>

    <h2>Capítulo 1.2</h2>
    <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Accusantium enim alias saepe, odio quod dicta. Odit veniam modi cumque, officiis sint aliquid, expedita nihil delectus fugit non recusandae corrupti minima.</p>
    
    <h1>Capítulo 2</h1>
    <h2>Capítulo 2.1</h2>
    <p>Lorem ipsum dolor sit, amet consectetur adipisicing elit. Neque ab in molestiae ut recusandae minima ipsum sequi officiis error obcaecati totam inventore, corrupti aspernatur pariatur natus porro explicabo repellendus animi.</p>
    <p style="text-align: right;"><a href="pagina02.html">Ir para a página 2</a></p><!--Apenas este parágrafo será movido para a direita-->

HARMONIA DE CORES

Círculo cromático:
Cores primárias: amarelo, vermelho, Azul

Cores secundárias: Laranja, violeta, verde

Cores terciárias: amarelo-esverdeado, amarelo-alaranjado, vermelho-alaranjado, vermelho-aroxeado, azul-aroxeado, azul-esverdeado
Obs:
Por convenção nós famos primeiramente a cor primária e depois as secundárias. As cores terciárias enquandram-se nos tons pasteis.

Dividindo o círculo ao meio, em um hemoisfério temos cores frias e do outro temos as quentes



Classicifações das cores buscando harmonia.

Paleta de cores: Uma paleta deve ter de 3 a 4 cores no máximo, além do branco e preto.

- Cores Complementares:
são aquelas que tem mais contraste entre si, para isso basta escolher a cor exatamento oposta no círculo cromático.

- Cores Análogas:
Ao contrário das cores complementares, são as cores que não tem tanto contraste. Para escolher, basta selecionar as cores vizinhas.
um bom exemplo é pegar  uma cor, duas cores Análogas e uma complementar

- Cores Análogas Relacionadas:
Não oferece tanto contraste.
uma cor, uma análoga, pula uma e pega a vizinha.

- Cores inter caladas:
Contraste mais duro.
pega uma, pula uma, pega outra, pula outra, pega outra.

- Cores Triádicas:
pega uma , pula 3, pega outra, pula 3, pega outra.

- Cores em quadrado:
Cores fortes e bem balanceadas.
pega uma, pula 2, pega uma, pula 2, pega outra, pula 2, pega uma.

- Cores tetrádicas:
pegar duas cores e seus respectivos complementares.

- Monocromia:
muda a saturação e a luminozidade (degradê)

- embedded-opentype
- Truetype-aat(Apple Advanced Typografy)
- svg

Obs: Não é recomendável deixar os arquivos de fontes na pasta principal, normalmente criamos uma pasta chamada fontes e colocamos todos os arquivos lá dentro.

Como inserir:

1°) inserir a fonte:
Ex:

<style>
    @font-face {
        font-family: 'xxx(qualquer nome)';
        src: url('O endereço da fonte');
    }
Obs: Também é possível inserir mais de um formato ao mesmo tempo. Para isso basta inserir ", " antes de ";" e um novo "url()"

2°) Chamar a fonte:
Ex:

    h1 {
        font-family: 'xxx', 'Times New Roman, Times, serif;
    }
</style>

Obs: Times New Roman foi inserido, pois caso o navegador não reconheça a fonte: 'xxx' o site não irá quebrar.

PESOS, ESTILOS E SHORTHAND FONT

https://www.youtube.com/watch?v=oHj5ez1bSkc&list=PLHz_AreHm4dlUpEXkY1AyVLQGcpSgVF8s&index=13

Para selecionar o peso da fonte, vamos utilizar no CSS o "font-weight". Esses ajustes variam a espessura da fonte, e podem ser:
- lighter, 
- normal, 
- bold, 
- bolder
- 100
- 200
- 300
- ...
- 900

Obs: Nem todas as fontes possuem todas as medidas bem como outras característica como "font-style: italic;..."
Ex:
<style>
    h1, h2{
        font-family: 'Times New Roman', Times, serif;
        font-weight: bolder;
        font-size: 3em;
        font-style: italic;
    }
</style>

Poderemos comprimir em uma única linha utilisando "shorthands". É preciso seguir a ordem de precedencia:
font-style => font-weight => font-size => font-family
Ex:
<style>
    h1{
        font: italic bolder 3em 'Times New Roman', Times, serif;
    }
</style>
Exatamente o mesmo código porém compactado, porém vale lembrar que nem todas as características precisam ser inseridas, mas as que forem precisam seguir ordem.

AS DIFERENÇAS ENTRE ID E CLASS:

https://www.youtube.com/watch?v=zXfTjPrMC_0&list=PLHz_AreHm4dlUpEXkY1AyVLQGcpSgVF8s&index=20

Segundo o W3C, dentro de um documento HTML só se pode ter um elemento por "id", se queremos que o estilo seja aplicado a mais de um elemento devemos usar o "class".
Ex:
1°-CSS
.avançado{
    color: rgb(172, 3, 3);
}
.destaque{
    background-color: rgba(214, 214, 111, 0.514);
}

2°-HTML:
<h2 class="avançado destaque">CSS Avançado</h2>

Obs: É possível também, inserir mais de uma "class", note que "avançado" e "destaque" não são separados por vírgula.

#principal{

    background-color: rgba(214, 214, 111, 0.514);
}

2°-HTML:
<h1 id="principal" class="destaque">Criando site com HTML e CSS</h1>
Obs: "id" e "class" não são separados por vírgula.

hierarquia:
1°- elemento
2°- .elemento(class)
3°- #elemento(id)

Obs: #principal também é um h1 porém por ser específico, precisa ser configurado depois do h1 puro, respeitando hierarquia. Vai herdar todas as características  do elementos acima.
