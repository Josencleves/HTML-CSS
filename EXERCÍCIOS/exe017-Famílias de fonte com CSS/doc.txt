TAMANHO DE FONTES E SUAS MEDIDAS

https://www.youtube.com/watch?v=NGfPXJGiNH8&list=PLHz_AreHm4dlUpEXkY1AyVLQGcpSgVF8s&index=12

Para selecionar o tamanho da fonte, vamos utilizar no CSS o "font-size". O "font-size" possue duas medidas.
-Medidas absolutas:
    cm, mm, in(polegada), px, pt, pc

-Medidas relativas:
    em(relativo ao tamanho do "M" maiúsculo de uma fonte), 
    ex(tamanho  do "x" minúsculo), 
    rem(Relativo ao tamanho da fonte), 
    vW(porcentagem da largura da tela), 
    vh(porcentagem da altura da tela),
    %
    recomendação da W3c usar: px, em

O tamanho de uma fonte é "1em" e por padrão é = 16px.

PESOS, ESTILOS E SHORTHAND FONT

https://www.youtube.com/watch?v=oHj5ez1bSkc&list=PLHz_AreHm4dlUpEXkY1AyVLQGcpSgVF8s&index=13

Para selecionar o peso da fonte, vamos utilizar no CSS o "font-weight". Esses ajustes variam a espessura da fonte, e podem ser:
- lighter, 
- normal, 
- bold, 
- bolder
- 100
- 200
- 300
- ...
- 900

Obs: Nem todas as fontes possuem todas as medidas bem como outras característica como "font-style: italic;..."
Ex:
<style>
    h1, h2{
        font-family: 'Times New Roman', Times, serif;
        font-weight: bolder;
        font-size: 3em;
        font-style: italic;
    }
</style>

Poderemos comprimir em uma única linha utilisando "shorthands". É preciso seguir a ordem de precedencia:
font-style => font-weight => font-size => font-family
Ex:
<style>
    h1{
        font: italic bolder 3em 'Times New Roman', Times, serif;
    }
</style>
Exatamente o mesmo código porém compactado, porém vale lembrar que nem todas as características precisam ser inseridas, mas as que forem precisam seguir ordem.

ALINHAMENTO:

https://www.youtube.com/watch?v=tJTtp4qyqdE&list=PLHz_AreHm4dlUpEXkY1AyVLQGcpSgVF8s&index=18


Relembrando a ordem de precedência do shorthand:
1°- font-weight
2°- font size.
3°- font family

Ex:
<style>
    body{
        font: normal 15px arial, verdana, serif;
        text-align: left;
    }
    h1{
        text-align: center;

    }
    h2{
        text-align: right;
    }
    p{
        text-align: justify;
        text-indent: 30px;
    }
</style>

Resumindo:
text align: left, start, right, justify.
text-indent: faz o avanço do parágrafo.
