IMAGENS DINÂMICAS


Com html utilizaremos 3 fotografias de tres tamanhos diferentes. Cada foto será mostrada dependendo do tamanho do dispositivo. para isso utilizaremos a tag <picture>
Ex:
<picture>
    <source media="(max-width: 750px)" srcset="imagens/foto-p.png" type="image/png">
    <source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png">
    <img src="./imagens/foto-g.png" alt="imagem flexível">
</picture>

Passo a passo: 
Usando a tag <img> e informando o endereço, vamos incluir primeiramente a foto maior que será exibida na tela maior. Depois vamos incluir acima da primeira, a próxima menor foto utilizando a tag <source: media: type> e atribuindo "(max-width: 777px)", o endereço da imagem e o "type"







REPRODUZINDO AUDIO.


Onde conseguir as músicas
youtube-painel de controle-youtube studio-biblioteca de audio- atriubição não necessária- usar filtros- baixar.
Os principais arquivos aceitos pelos navegaroes são:
1°- mp3 
2°- wav (Evitar o formato devido ao tamanho)
3°- ogg

Para inserir arquivo de audio, utilizaremos a tag "<audio>" 
Ex-01:

<audio src="endereço do arquivo" controls(para aparecer os comandos de play,stop,volume...) ></audio>

Ex-02:

<audio preload="metadata" autoplay controls loop>
    
    <source src="./midia/White River - Aakash Gandhi.mp3" type="audio/mpeg" >
    
    <source src="./midia/White River - Aakash Gandhi.ogg" type="audio/ogg">
    
    <source src="./midia/White River - Aakash Gandhi.wav" type="audio/wav">
    
    <p>Infelismente seu navegador não consegue reproduzir àudio. <a href="./midia/White River - Aakash Gandhi.mp3">Clique aqui para baixar o arquivo</a></p>
</audio>

Desa forma o código vai tentar abrir cada formato até encontrar um que funcione e caso nenhum funcione, abrirar o link para download do arquivo.
Obs:
    1- O atributo "preload" possue 3 valores: auto, metadata e none
    Auto = ele só vai considerar que o site carregou depois de carregar todo o audio

    metadata = Vai carregar apenas os metadados. nome do arquivo, formatos...

    none = não carrega nada. Só começa quando o usuário apertar o play

    2- O atributo autoplay não funciona pelo google chrome

    3- O atributo "loop" faz o audio recomeçar quando chega ao final.
