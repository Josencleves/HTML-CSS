O mais indicado método para estilo é criando um documento CSS atravé da tag <link> no <head>.

Ex:
<html>
<link rel="stylesheet" href="style(ou qualquer outro título que voce escolheu para sua pasta CSS).css">
</html>
depois cria-se uma arquivo CSS onde iremos inserir todos os estilos que serão soliciados para o "html" atravé do "<link>"

Ex:
<CSS>

@charset "UTF8";
body {
    background-color: lightgoldenrodyellow;
    font-family: Arial, Helvetica, sans-serif;
}
h1{
    color:  maroon;
    background: darkgoldenrod
}
h2{
    color: darkred;
}
p{
    text-align: justify;
}

Vantagem:
todos os estilos são aplicados aos elementos em todas as paginas e ainda é possível utilizar CSS interno e CSS inline  para páginas e elementos específicos.
Ex:
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Estilos Externos</title>
    <link rel="shortcut icon" href="favicon.icon" type="image/x-icon">
    <link rel="stylesheet" href="style.css">
    <style>
        h1{
            text-decoration: underline;
        }
    </style>
    <!--Estilo aplicado apenas na página em questão-->
</head>
<body>
    <h1>Capítulo 1</h1>
    <h2>Cpítulo 1.1</h2>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Eaque, maxime inventore! Voluptate, totam! Dolorum ut iusto et, ipsa provident distinctio minima eaque nihil doloremque commodi, quaerat tempora facere nam modi.</p>

    <h2>Capítulo 1.2</h2>
    <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Accusantium enim alias saepe, odio quod dicta. Odit veniam modi cumque, officiis sint aliquid, expedita nihil delectus fugit non recusandae corrupti minima.</p>
    
    <h1>Capítulo 2</h1>
    <h2>Capítulo 2.1</h2>
    <p>Lorem ipsum dolor sit, amet consectetur adipisicing elit. Neque ab in molestiae ut recusandae minima ipsum sequi officiis error obcaecati totam inventore, corrupti aspernatur pariatur natus porro explicabo repellendus animi.</p>
    <p style="text-align: right;"><a href="pagina02.html">Ir para a página 2</a></p><!--Apenas este parágrafo será movido para a direita-->
</body>

Temos o <link> para CSS externo, um <style> aplicado a todos os "h1" apenas na página principal e CSS inline apenas no  ultimo elemento <p>
